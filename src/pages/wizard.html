Send to Feishu is a browser extension that helps you send text and image content to Feishu by simply using the context menu.

We just need to set up a few things before you can start using it.

We need to create a Feishu App ðŸš€
Don't worry, it's super easy and free!

Just follow the steps below:

  1. Go to the Feishu Open Platform.
  2. Click on "Create Custom App".
  3. Fill in the necessary information for your app.
  4. After creating the app, go to "Credentials & Basic Info" in the app's dashboard.
  5. You'll find your App ID and App Secret here. Keep these handy!
  6. Important: Make sure to enable the necessary permissions for your app, such as sending messages.

Now, we need to get your Feishu chat ID.

It's the identifier for the chat where you want to send messages.

Here's how to obtain that:

  1. Open Feishu and go to the chat (group or personal) where you want to send messages.
  2. Click on the chat name at the top to open chat settings.
  3. Scroll down and find "Copy Chat ID".
  4. Click to copy the chat ID and that's all we need!

Note: If you don't see "Copy Chat ID", you may need to enable developer mode in Feishu settings.

All we have to do now is save these credentials to the extension.
