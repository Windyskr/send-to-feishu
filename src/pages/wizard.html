Send to Telegram is a browser extension that helps you send text and image content to Telegram by simply using the context menu.

We just need to set up a few things before you can start using it.

We need to create a Telegram Bot ðŸ¤–
Don't worry, it's super easy and free!

Just follow the steps below:

  1. Open the Telegram app.
  2. In the search bar, search for @BotFather.
    Make sure it has the verified badge
  3. Click on the search result to start chatting.
  4. Click on the Start button.
  5. Type /newbot to the chat.
  6. Give a name to your bot.
  7. Set a username (must end with bot).
  8. After the bot is created, open the chat dialog with it and type or click on /start.
  9. Return to the chat with BotFather, copy the token and that's it! ðŸ¥³

Starts with a Chat ID ðŸ˜‚

Now, we need to get your Telegram chat ID.

It's your profile identifier and a necessity for the bot to be able to send you messages.

Here is how to obtain that:

  1. Open the Telegram app.
  2. In the search bar, search for @userinfobot
  3. Click on the search result to start chatting.
  4. Click on the Start button.
  5. Get the chat ID and that's all we need!

All we have to do now is saving these credentials to the extension.
